A Choice Exception Strategy handles an exception based on the message's content. It catches the exception, checks the exception type using a MEL expression and content of the message, and then maps the exception to appropriate exception strategy.
It's like multiple catch blocks in Java.

A Choice Exception Strategy cannot have another Choice Exception Strategy inside it.

Configuring a Choice Exception Strategy

The same flow from Catch Exception Strategy will be used.
  • Drag the Choice Exception Strategy into the Error Handling area of the flow
