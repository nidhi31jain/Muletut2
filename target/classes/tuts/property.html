Property transformer can be used to set, remove or copy outbound properties.

Inbound properties are immutable.

Configuration

  • To set a property, open the properties tab of property transformer.
  • Select the Set property radio button. Give the name and value for the property.
  • Value can be a MEL expression also.
  • To acccess the property, use #[message.outboundProperties['size']]
  • To remove a property, just select the remove property radio button and specify the name of property to remove.
