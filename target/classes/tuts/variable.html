Variables, as known to everybody, are used to save some value.
Mule have 3 type of variables
  • Flow variables
  • Session variables
  • Record variables (Will be discussed in batch processing)

Flow Variable

Flow variables are tied to the message as it moves in a flow. When the message leaves the flow, the flow variables cease to exist.The scope of flow variables is limited to the flow only. A variable transformer is used to set or remove flow variables.
MEL expressions can also be given as value like #[message.inboundProperties.'http.listener.path'] .
Flow variables is accessed using #[flowVars.myVar]

Don't write the value in quotes as it will become part of the value too.

Session Variable

Use a session variable transformer to set or remove a session variable which is tied to the message for its entire life cycle, across multiple flows.
Session variables is accessed using #[sessionVars.mySessionVar]

Session variables can be propagated using VM or flow references, but not HTTP connectors.

The variable value is String by default, to convert it to integer use MEL expression. Write like #[1].
