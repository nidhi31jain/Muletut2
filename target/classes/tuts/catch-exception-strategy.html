Catch Exception strategy can be used to customize the error handling in mule. It catches all exception thrown in the flow.

Unlike java, catch exception strategy cannot throw a new exception.

Configuring Catch Exception Strategy

  • Include a HTTP component and configure to listen on exception path
  • Drag a Groovy Transformer
  • Open the Script Transformer's properties tab and write throw new IllegalArgumentException('exception')
  • In the Error Handling section, drag a Catch Exception Strategy and include a logger inside it
  • On running the flow, it will throw an exception and the logger inside Catch Exception Strategy will be executed
