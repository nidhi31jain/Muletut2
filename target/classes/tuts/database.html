The Database Connector can connect to almost any Java Database Connectivity (JDBC)-compliant relational database.
Operations supported are but not limited to
  • SELECT, INSERT, UPDATE, DELETE
  • Bulk Execute
  • Stored Procedures
  • Data Definition Language(DDL) requests
It offers out of the box support for Oracle, Derby and MySQL. All others are supported by a generic database configuration.

Configuring a Database Connector

  • Include the DB connector in your flow and double click it.
  • Create a new configuration by clicking green plus sign or edit an existing one.
  • Choose a type of global element from the next screen.
  • For MySQL database connection, fill out the details.
  • Add the required MySQL jar for the connection
  • The test connection should be successful now.
  • Back in the properties tab, select an operation

  • Here, parameterized query is selected. Mule gives the option of dynamic query and from template also.
  • In parameterized query, mule replaces the MEL expression with expression's value.
