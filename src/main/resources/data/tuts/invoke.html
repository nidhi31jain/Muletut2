The Invoke Component is used to invoke a java method. The Java object is defined in a Spring bean. It can take an array of argument expressions. Mule maps the request to the method specified and the number of arguments.
  • Before proceeding, create a Java class.
    					
package com.muletut;

public class FileDetails {
 public String getFileDetails(String fileName, String fileDirectory ){
	 return fileName+" "+fileDirectory;
 }
}
  • Include a File component, an Invoke component and a logger in the flow.
  • Configure the File connector
  • Go to Global Elements tab, and click on Create
  • In the Choose Global Type screen, select Bean under Beans
  • Give the name for the bean and enter the class name to invoke. Click OK

    Name of the bean can be different from the Java class

  • Open the properties tab for Invoke component
  • Enter bean name in front of Object ref. It is the reference to the bean to be invoked.
  • Enter method name to be invoked. Also, enter the arguments to be passed.
  • Run the project
