• Open Anypoint Studio
  • Go to File > New > Mule Project
  • Drag and drop an HTTP connector from palette onto canvas
  • Drag and drop a set payload component after HTTP connector
  • Double click the HTTP connector to open its properties tab
  • Click the green plus sign to configure the connector settings
  • Enter the path
  • Open the properties tab for Set Payload component.
  • Set the value field to Hello World
  • Right click on the canvas and select Run project firstproject
  • Open the web browser and navigate to the URL
