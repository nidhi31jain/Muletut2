HTTP connector allows a mule application to send and receive HTTP or HTTPS request given a host, port and path.
The Connector can be used as either
  • Listener: To listen to request coming to the application.
  • Requester: To send request to certain address from the application and get the response.

HTTP listener

  • Place the HTTP connector in source section.
  • Add a connector configuration.
  • Back in the properties tab, add path. It is the URI to listen on

HTTP Requester

  • Place the HTTP connector in message processor section.
  • Add a connector configuration.
  • Back in the properties tab, add path. It is the URI to send the request to
  • Select the method from the dropdown.
