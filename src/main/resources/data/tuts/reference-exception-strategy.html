This strategy can be used to refer to one of the global exception strategy. When implemented, the flow is asked to refer to that exception strategy for handling errors.

Creating Global Exception Strategy

A global exception strategy can be created to reuse in flows.
  • Move the Catch Exception Strategy outside the flow
  • Go to Global Elements and create a new global configuration
  • Drag a Reference Exception Strategy to the Error Handling section of the flow and make it refer to the global exception strategy
