The choice router routes message based on a condition. It's similar to if then else block. It uses MEL expressions to evaluate the condition and route based on that, if no condition is true than it routes to the default route.

Using a Choice Router

  • Drag the choice router into your flow. Include two loggers.
  • Open the properties tab for choice router and write the MEL expressions to define routing logic.
