Dataweave selectors are used to navigate through the Objects or Arrays, sometimes in combination with operators

Single Value Selector

Returns the first value whose key matches the expression.
  • Input
    					
<persons>
<Name>Elena</Name>
<Name>Alena</Name>
</persons>
  • Transform
    					
%dw 1.0
%output application/json
---
Name: payload.persons.Name
  • Output
    					
{
  "Name": "Elena"
}

Multi Value Selector

Returns all the values whose key matches the expression.
  • Transform on same input as Single Value Selector
    					
%dw 1.0
%output application/json
---
Name: payload.persons.*Name
  • Output
    					
{
  "Name": [
    "Elena",
    "Alena"
  ]
}

Index Selector

Returns the element at the specified index.
If index is positive or 0, it starts counting from beginning else from end where -1 is the last element
  • Transform on same input as Single Value Selector
    					
%dw 1.0
%output application/json
---
Name: payload.persons[1]
  • Output
    					
{
  "Name": "Alena"
}

Descendants Selector

Returns the element at the specified index.
If index is positive or 0, it starts counting from beginning else from end where -1 is the last element
  • Transform on same input as Single Value Selector
    					
%dw 1.0
%output application/json
---
Name: payload..Name
  • Output
    					
{
  "Name": [
    "Elena"
  ]
}
