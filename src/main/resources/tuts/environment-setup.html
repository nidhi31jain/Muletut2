Anypoint Studio, an eclipse based IDE, is a graphical integration design environment for Mule applications. Some features of Anypoint are:
  • Graphical design support
  • In-built Mule server
  • Pre-built connectors to connect to many popular services (Salesforce, )
  • Supports standard protocols like HTTP, HTTPS, FTP, SFTP and more
  • Both, Graphical and XML are available

Installing Anypoint Studio

  • Make sure Java is installed
  • Download Anypoint Studio
  • Fill up the details on the next page.
  • Unzip the zip file
  • Start studio by clicking the Anypoint Studio icon in the directory
  • Select the workspace for working or create a new folder for the purpose
